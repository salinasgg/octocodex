Implementación de Logout en Dashboards

1. Para admin/dashboard.php:

<!-- En el header o navbar del dashboard -->
<div class="navbar-nav ms-auto">
    <button id="logout-btn" class="btn btn-danger">
        <i class="fas fa-sign-out-alt me-2"></i>Cerrar Sesión
    </button>
</div>

<!-- Al final del archivo, antes de </body> -->
<script src="../js/logout.js"></script>
2. Para usuario/dashboard.php:

<!-- En el header o navbar del dashboard -->
<div class="navbar-nav ms-auto">
    <button id="logout-btn" class="btn btn-danger">
        <i class="fas fa-sign-out-alt me-2"></i>Cerrar Sesión
    </button>
</div>

<!-- Al final del archivo, antes de </body> -->
<script src="../js/logout.js"></script>
3. Archivo js/logout.js (ya configurado):

$(document).ready(function() {
    $('#logout-btn').click(function() {
        $.ajax({
            url: '../php/logout.php',
            type: 'POST',
            dataType: 'json',
            success: function(response) {
                if (response.success) {
                    // Usar la URL de redirección que viene del servidor
                    window.location.href = response.redirect;
                } else {
                    // Si falla, redirigir directamente
                    window.location.href = '../php/logout.php';
                }
            },
            error: function() {
                // Si hay error en AJAX, redirigir directamente
                window.location.href = '../php/logout.php';
            }
        });
    });
});
4. Rutas de redirección:
  • Desde admin/dashboard.php: → ../php/logout.php → ../index.php = condor/index.php ✅
  • Desde usuario/dashboard.php: → ../php/logout.php → ../index.php = condor/index.php ✅
  • Desde dashboard.php: → php/logout.php → ../index.php = condor/index.php ✅
5. Prueba del logout:
Para probar:
  1. Haz login como administrador o usuario
  2. Ve al dashboard correspondiente
  3. Haz clic en "Cerrar Sesión"
  4. Deberías ser redirigido a condor/index.php
