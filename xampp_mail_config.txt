CONFIGURACIÓN DE CORREOS EN XAMPP
====================================

Para que el formulario de contacto funcione correctamente, necesitas configurar XAMPP para el envío de correos.

OPCIÓN 1: CONFIGURAR SMTP LOCAL
--------------------------------

1. Edita el archivo: C:\xampp\php\php.ini

2. Busca la sección [mail function] y configura:
   [mail function]
   SMTP = localhost
   smtp_port = 25
   sendmail_from = noreply@devstudio.com

3. Instala un servidor SMTP local como:
   - Mercury (incluido en XAMPP)
   - Postfix
   - hMailServer

OPCIÓN 2: USAR GMAIL SMTP
--------------------------

1. Edita el archivo: C:\xampp\php\php.ini

2. Configura:
   [mail function]
   SMTP = smtp.gmail.com
   smtp_port = 587
   sendmail_from = tu-email@gmail.com

3. Habilita autenticación de 2 factores en Gmail
4. Genera una contraseña de aplicación

OPCIÓN 3: USAR SERVICIO DE TERCEROS
------------------------------------

1. Mailgun
2. SendGrid
3. Amazon SES
4. Mailtrap (para pruebas)

OPCIÓN 4: SOLUCIÓN TEMPORAL
----------------------------

Si no puedes configurar SMTP, el formulario guardará los mensajes en:
- email_log.txt (envíos exitosos)
- email_error_log.txt (errores)

Y mostrará un mensaje de éxito al usuario.

CONFIGURACIÓN RÁPIDA PARA PRUEBAS
=================================

1. Descarga Mercury desde: https://www.pmail.com/
2. Instálalo en C:\xampp\Mercury\
3. Configura el dominio como: localhost
4. Configura php.ini para usar Mercury

REINICIAR SERVICIOS
===================

Después de cualquier cambio:
1. Detén Apache en XAMPP Control Panel
2. Inicia Apache nuevamente
3. Prueba el formulario

VERIFICACIÓN
===========

Para verificar que funciona:
1. Envía un mensaje desde el formulario
2. Revisa email_log.txt
3. Revisa tu bandeja de entrada en salinasgeganb@gmail.com
