Soporte Técnico

Se atendia llamada del usuario de internet de Arnet mediante el aplicativo Avaya haciendo una breve presentacion consultando datos personales para corroborar la identidad del mismo.

El motivo del llamado podia ser por diversas razones, falta de servicio de internet, micro cortes, inconvenientes con el Outlook, se realizaba un sondeo para poder indentificar realmente cual era el problema realizando una serie de preguntas como por ej: si estaba frente a la PC para agilizar la atencion, que luces tenia encendida del modem, que modem tenia.

Seguido de una serie de acciones con el fin de normalizar el servicio, en caso de que el inconveniente exceda nuetra capacidad, levantabamos un Ticket en un aplicativo (SUR) para que lo pueda tomar Soporte nivel 2.

Herramientas
AVAYA Con la cual se atendian los llamados.
ADSL-test Con el que podiamos verificar informacion tecnica del host a la cual estaba conectada su linea telefonica y asi poder resetear el puerto y poder reestablecer la señal del modem
CMS Informacion tecnica de la linea fija
SUR Sistema unificado de reclamos, por el cual levantabamos ticket hacia nivel 2.
UTOPIA Sobre el cual corroborabamos la identidad del usuario, reseteo de clave, cambio nombre de usuario de conexion.

Habilitaciones

El usuario se comunicaba por que ya habia recibido el kit de instalacion del modem.

El motivo del llamado era para consultar como se realizaba la conexion del modem y telefono utilizando el splitter que bifurcaba la señal telefonica de la de ADSL ó por que ya tenia todo conectado y queria comenzar a navegar.

En caso de tener todo conectado y el modem correctamente las luces encendidas con señal de internet, se procedia a realizar una serie de patos para poder crear su propio usuairo y clave, conectar, antes de finalizar la comunicacion se le indicaba abrir explorador de internet y tipear una web para corroborar que efectivamente se encontraba habilitado, conectado y navegando.

Visita Técnica a domicilio

Nos comunicabamos con el usuario para pactar una visita a domicilio.

Esto se realizaba si el usuario no habia podido conectar correctamente el modem ó no sabia como hacerlo y solicitaba técnico.

Para cambio de modem luego de verificar que el mismo ya no funcionaba ó si habia tenido algun inconveniente en el hogar que haya provocado la rotura del mismo.

Coordinando con el usuario se pactaba la visita en tres rangos horarios y fecha deseada por el usuario sujeta a disponibilidad en nuestros sistemas
